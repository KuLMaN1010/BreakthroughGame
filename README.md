Overview
This project implements intelligent agents to play the classic board game, Breakthrough.

Implemented the agents using powerful search algorithms: Minimax and Alpha-Beta pruning.

Designed and tested multiple heuristics (offensive, defensive, hybrid) to optimize agent decision-making and strategy.

Evaluated performance through extensive self-play matchups, demonstrating a strong balance of efficiency, strategic depth, and compelling gameplay visualization.

Installation
Follow these steps to build the project and install dependencies:

  Clone the repository: > git clone https://github.com/KuLMaN1010/BreakthroughGame
  
  Navigate to the project directory: > cd BreakthroughGame
  
  Install dependencies using Conda: > conda env create -f conda.yml
  This command reads the environment details from the conda.yml file.

Usage
To start the game or run simulations:

Using Conda:
  conda activate {venv}
  python {entrypoint}

Note: Replace {venv} with your Conda environment name and {entrypoint} with the main script name (e.g., main.py).
